# 💱 Конвертер валют

Конвертер всех распространённых мировых валют с **точными курсами Банка Израиля (BOI)**
и **Европейского центрального банка (ЕЦБ / банки Европы)**.

## Возможности

- 36 распространённых валют (USD, EUR, ILS, GBP, CHF, JPY, CNY, RUB, UAH, PLN и др.)
- Переключение источника курса: **🇪🇺 ЕЦБ** или **🇮🇱 Банк Израиля**
- Живые курсы загружаются прямо в браузере:
  - **ЕЦБ** — через [Frankfurter API](https://www.frankfurter.app) (официальные референсные курсы ЕЦБ)
  - **Банк Израиля** — через официальный [Public API BOI](https://boi.org.il)
- Быстрый просмотр курса выбранной валюты к популярным
- Кнопка «поменять местами» и мгновенный пересчёт
- Резервная таблица курсов на случай отсутствия сети — приложение работает всегда

## Запуск

Это статичный сайт без сборки. Откройте `index.html` в браузере либо запустите
локальный сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

> Курсы загружаются в браузере пользователя. Для запросов к API нужен доступ
> в интернет; при его отсутствии используется встроенная резервная таблица.

## Структура

| Файл         | Назначение                                   |
|--------------|----------------------------------------------|
| `index.html` | Разметка интерфейса                          |
| `styles.css` | Оформление                                   |
| `rates.js`   | Список валют и резервная таблица курсов       |
| `app.js`     | Логика: загрузка курсов, конвертация, события |

## Источники курсов

- Европейский центральный банк (референсные курсы евро) — `api.frankfurter.app`
- Банк Израиля (официальные представительные курсы) — `boi.org.il/PublicApi/GetExchangeRates`

Курсы носят справочный характер.
